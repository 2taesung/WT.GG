시연 스크립트

회원가입/로그인 기능 무

1. 메인 페이지
- 우측 상단에 햄버거 바
- 중앙에 메인 메뉴 (랭킹, 커뮤니티, 추천)
  메인 메뉴 클릭 후 아래 버튼으로 이동 가능

2. 랭킹 페이지
- 웹툰들의 랭킹이 나열되고 클릭시 각 웹툰의 상세페이지로 이동
  우측 상단의 스크롤바를 통해 장르별 랭킹 기능 제공

3. 상세 페이지
- 해당 웹툰에 대한 페이지와 해당 웹툰 사이트로 가는 보러가기 버튼, 우측의 팬 게시판 이동 버튼
- 하단에 해당 웹툰에 대한 통합 평점
- 하단에 해당 웹툰의 성향에 대한 지수들
